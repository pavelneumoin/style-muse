## 📖 О проекте

**«Детский Погодник»** — это образовательный детский проект, созданный для того, чтобы научить детей дошкольного и младшего школьного возраста:

- 🌡️ **Понимать погоду** — что такое температура, влажность, ветер
- � **Одеваться самостоятельно** — выбирать одежду по погоде
- 🔬 **Интересоваться наукой** — узнавать о погодных явлениях через AI-помощника
- 🎨 **Развивать творчество** — экспериментировать с цветами одежды на аватарах

Проект использует современные технологии искусственного интеллекта (YandexGPT), чтобы объяснять сложные понятия простым, понятным для детей языком с использованием эмодзи и весёлых примеров.

---

## 👨‍👩‍👧‍👦 Зачем нужен этот проект?

Многие дети не понимают, почему нужно одеваться теплее, когда холодно, или почему важно взять зонтик. **«Детский Погодник»** решает эту проблему:

1. **Наглядность** — ребёнок видит аватар в одежде, соответствующей погоде
2. **Интерактивность** — можно нажимать на элементы и получать объяснения от AI
3. **Обучение** — нейросеть объясняет погодные явления простым языком
4. **Игра** — можно менять цвета одежды на аватарах

## 🚀 Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/pavelneumoin/style-muse.git
cd style-muse
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения
Создайте файл `.env` в корне проекта:
```env
YANDEX_FOLDER_ID=ваш_folder_id
YANDEX_API_KEY=ваш_api_key
OPENWEATHER_API_KEY=ваш_ключ_openweather
```

### 4. Запустите приложение
```bash
python app.py
```

### 5. Откройте в браузере
```
http://127.0.0.1:5000
```
