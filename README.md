# 🌤️ Детский Погодник

> Умный помощник для детей, который рассказывает о погоде и помогает выбрать одежду!

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Flask](https://img.shields.io/badge/Flask-2.0+-green)
![YandexGPT](https://img.shields.io/badge/AI-YandexGPT-yellow)

## ✨ Возможности

- 🌡️ **Погода в реальном времени** — температура, влажность, ветер
- 👗 **Интерактивные аватары** — мальчик и девочка с одеждой по погоде
- 🎨 **Выбор цвета одежды** — 45+ цветов в палитре
- 🤖 **AI-помощник** — объясняет погодные явления простым языком для детей
- 🧲 **Магнитные бури** — отслеживание геомагнитной активности
- 💡 **Советы по одежде** — генерируются нейросетью YandexGPT

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/pavelneumoin/style-muse.git
cd style-muse
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения
Создайте файл `.env`:
```env
YANDEX_FOLDER_ID=ваш_folder_id
YANDEX_API_KEY=ваш_api_key
OPENWEATHER_API_KEY=ваш_ключ_openweather
```

### 4. Запустите приложение
```bash
python app.py
```

Откройте http://127.0.0.1:5000 в браузере 🎉

## 📸 Скриншоты

Приложение показывает:
- Текущую погоду с эмодзи
- Двух персонажей в одежде по сезону
- AI-советы для детей

## 🔧 Технологии

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **AI:** YandexGPT API
- **Погода:** OpenWeatherMap API
- **Магнитные бури:** NOAA API
